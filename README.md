Building a professional IT HomeLab step by step.
---
## Project Overview

This repository documents my journey of building a professional IT HomeLab from scratch.

The project focuses on designing and implementing a realistic enterprise environment using virtualization, Windows Server, Linux, networking and automation technologies.

Instead of publishing only the final results, every important decision, configuration, challenge and solution is documented step by step.

The goal is to create an open technical reference for students, IT enthusiasts and system administrators while also serving as my professional portfolio.
